## 摘要
现有的基于视觉-语言模型（VLM）的AI生成图像质量评估（AIGIQA）方法存在一个根本性的语义-失真维度冲突：为语义区分优化的单一表征必然地将组合理解与低级感知敏感度纠缠在一起，导致其对细粒度质量退化视而不见。我们提出MST-CLIPIQA，一种多尺度双流框架，通过显式的表征解耦实现层次化的视觉-语言对齐。我们的架构利用具有互补块粒度的双CLIP编码器：粗粒度流捕捉全局语义连贯性，而细粒度流保留纹理特征和伪影模式。一种受信息瓶颈启发的门控融合机制执行自适应跨尺度蒸馏，当生成提示可用时，可选的交叉注意力实现锚定于提示的对应关系评估。在五个基准上的广泛实验取得了新的最先进结果，在质量预测上平均提升1.11% SRCC，在文本-图像对应预测上平均提升2.35% SRCC，同时仅需0.8M可训练参数即可保持效率。我们的项目可在https://github.com/YMlinfeng/MST-CLIPIQA获取。

## Abstract
Existing vision-language model (VLM)-based AI-generated image quality assessment (AIGIQA) methods suffer from a fundamental semantic-distortion dimensional conflict: monolithic representations optimized for semantic discrimination inherently entangle compositional understanding with low-level perceptual sensitivity, rendering them blind to fine-grained quality degradations. We introduce MST-CLIPIQA, a multi-scale two-stream framework that achieves hierarchical vision-language alignment through explicit representational decoupling. Our architecture leverages dual CLIP encoders with complementary patch granularities: coarse-grained streams capture global semantic coherence while fine-grained streams preserve textural signatures and artifact patterns. An information bottleneck-inspired gated fusion mechanism performs adaptive cross-scale distillation, with optional cross-attention enabling prompt-anchored correspondence evaluation when generation prompts are available. Extensive experiments across five benchmarks establish new state-of-the-art results, achieving average improvements of 1.11 percent SRCC on quality and 2.35 percent SRCC on text-image correspondence prediction, while maintaining efficiency with only 0.8M trainable parameters. Our project is available at https://github.com/YMlinfeng/MST-CLIPIQA.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：现有基于视觉-语言模型（VLM）的 AI 生成图像质量评估（AIGIQA）方法存在 **语义-失真维度冲突**——为语义区分优化的单一表征必然将组合理解与低级感知敏感度纠缠在一起，导致模型对细粒度质量退化（如纹理伪影、局部缺陷）视而不见。
- **核心挑战**：
  - 人类质量感知是层次化的：粗粒度关注全局语义，细粒度检测局部失真，但单尺度视觉表征牺牲了在两个空间尺度上的敏感性。
  - 多尺度特征提取引入跨尺度信息冗余，简单融合会稀释质量判别信号。
  - AIGI 特有的生成语义错位（如解剖不合理、物理不可能、提示-内容不符）需要显式跨模态推理。
- **意义**：该工作旨在通过显式解耦语义和失真表征，实现层次化的视觉-语言对齐，从而提升 AIGI QA 的全面性和准确性。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
- 采用 **多尺度双流架构（MSTFE）**，利用双 CLIP 编码器分别处理粗粒度语义和细粒度纹理，实现表征解耦。
- 通过 **信息瓶颈启发的门控融合（GFF）** 自适应整合多尺度信息，过滤冗余。
- 当生成提示可用时，增加 **交叉注意力模块（Cross-Attention）** 进行提示锚定的对应关系评估。

### 关键技术细节
- **多尺度双流特征提取**：
  - 粗粒度流（patch size = 32×32）→ 全局语义连贯性。
  - 细粒度流（patch size = 16×16）→ 局部纹理和伪影模式。
  - 使用同一个 CLIP ViT-B/32 编码器（冻结），通过修改 tokenization 的 patch 大小实现双流。
- **门控融合（GFF）**：
  - 计算门控系数 `g = σ(W_g · ReLU(W_h [f_c; f_f] + b_h) + b_g)`，实现逐维度插值 `z = g ⊙ φ_c(f_c) + (1-g) ⊙ φ_f(f_f)`。
  - 实现信息瓶颈近似：自动选择每个维度中更质量相关的尺度，丢弃冗余。
- **提示锚定交叉注意力**（仅 MST-CLIPIQA*）：
  - 使用冻结 CLIP 文本编码器提取提示 token 嵌入，以融合视觉表示 z 作为 query，文本嵌入作为 key/value，计算交叉注意力，输出通过学习标量 α 残差融合。
- **回归头**：包含残差连接的 MLP（GELU 激活），输出质量得分。
- **损失函数**：组合损失 `L = L_MSE + λ L_rank`，平衡均方误差和排序一致性。
- **参数**：仅训练 ~0.8M 参数（门控网络、投影层、回归头），视觉编码器冻结。

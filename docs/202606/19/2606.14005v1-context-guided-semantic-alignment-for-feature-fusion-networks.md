## 摘要
特征融合网络是现代目标检测器中的基本组件，通过聚合多尺度特征来检测不同大小的目标。然而，直接融合来自不同金字塔层级的特征往往会因表示异构而导致语义不一致。本文提出特征交互网络（FINE），这是一种轻量级的语义对齐模块，通过跨层级注意力在融合前利用高层级上下文引导来细化低层级特征。为了弥合结构差异并确保计算效率，我们引入了对齐感知令牌采样，该采样对齐跨尺度的对应空间区域，将注意力复杂度降低一个数量级。由此产生的注意力权重生成一个空间-通道调制图，通过上采样并通过残差逐元素调制应用于低层级特征。该机制确保网络能够选择性地增强语义相关像素，同时保持密集预测任务所需的亚像素定位精度。FINE 可广泛应用于各种检测器，在不牺牲效率的前提下持续提升检测精度。

## Abstract
Feature fusion networks are fundamental components in modern object detectors, aggregating multi-scale features to detect objects of varying sizes. However, directly fusing features from different pyramid levels often introduces semantic inconsistency due to their heterogeneous representations. In this paper, we propose Feature Interaction NEtwork (FINE), a lightweight semantic alignment module that refines low-level features via high-level contextual guidance using cross-level attention prior to fusion. To bridge the structural gap and ensure computational efficiency, we introduce an Alignment-Aware Token Sampling that aligns corresponding spatial regions across scales, reducing the attention complexity by an order of magnitude. The resulting attention weights generate a spatial-channel modulation map that is upsampled and applied to the low-level features via residual element-wise modulation. This mechanism ensures that the network selectively enhances semantically relevant pixels while preserving the sub-pixel localization accuracy necessary for dense prediction tasks. FINE is generally applicable to various detectors and consistently improves detection accuracy without compromising efficiency.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：在目标检测的特征金字塔网络中，不同层级（低层与高层）的特征在语义表示上存在严重的**语义不一致**（semantic inconsistency）。低层特征保留空间细节但缺乏语义抽象，高层特征语义丰富但空间粗糙。直接通过逐元素加法或拼接进行融合，会强制位置对齐而未解决表示差异，导致信息冲突，增加误分类（尤其在背景杂乱中），并损害小目标检测性能。
- **背景**：现代检测器采用骨干-颈部-头部结构，颈部负责多尺度特征融合（如FPN、PANet等）。现有工作尝试通过通道重标定、重新设计融合结构或学习上采样偏移来缓解，但要么计算量大，要么缺乏即插即用性。注意力机制被广泛用于建模依赖关系，但跨层级密集注意力的计算成本过高，且由于有效感受野（ERF）不一致而无法建立可靠对应。
- **整体含义**：提出**FINE（Feature Interaction NEtwork）**——一个轻量级、即插即用的语义对齐模块，在融合前利用高层上下文通过跨层注意力引导来精炼低层特征，以较低代价缓解语义不一致，提升各类检测器精度。

## 2. 方法论

### 核心思想
在标准融合过程（`Flow`与上采样后的`Fhigh`相加）之前，先用FINE将`Flow`替换为经高层上下文调制后的对齐特征`F_aligned_low`。

### 关键技术细节

1. **对齐感知令牌采样（Alignment-Aware Token Sampling, AATS）**  
   - 由于相邻层级特征的ERF不匹配（高层步幅更大），且直接像素对像素注意力计算昂贵，AATS通过非对称下采样对齐ERF并压缩令牌数。  
   - 对高层特征使用核大小`k`的平均池化下采样，对低层特征使用核大小`r·k`的下采样，其中`r`为对齐采样比率（默认设为2，与骨干步幅比匹配）。  
   - 例如：`F_high`下采样到`H_high/k × W_high/k`，`F_low`下采样到`H_low/(r·k) × W_low/(r·k)`。下采样算子采用无参数平均池化（消融表明效果与可学习算子相当）。

2. **瓶颈多头部跨层注意力（Bottleneck Multi-Head Cross-Level Attention）**  
   - 将下采样后的`F_low`展平作为查询（Q），下采样后的`F_high`展平作为键（K）和值（V），进行标准点积注意力。  
   - 使用多头机制（h个头，每头维度d=C/h），在多个子空间中捕捉不同语义交互，默认设置每头维度d=16。  
   - 输出经拼接、投影、重塑后得到粗尺度的调制图`M_hat`，然后上采样`r·k`倍恢复到`Flow`原始分辨率，得到调制图M。

3. **残差空间-通道调制（Residual Spatial-Channel Modulation）**  
   - 调制图M提供逐位置、逐通道的缩放因子，但因其来自粗尺度估计，直接覆盖会丢失细粒度细节。  
   - 采用残差形式：`F_aligned_low = (Flow ⊙ M) + Flow`，其中⊙为逐元素乘法。这样既通过乘法选择性增强语义区域，又保留原始高分辨率特征信息。  
   - 相比直接调制或直接使用注意力输出作为新特征，残差调制更有效地平衡语义对齐与定位精度。

## 6. 主要结论与发现

- FINE在几乎所有检测器上带来了一致的mAP提升（+0.5~+2.1 AP），且计算开销极小（FLOPs增加约1~2G，参数增加约1M）。  
- 提升主要由**误检（FP）减少**（-9.93%）和**小目标检测**改善驱动（APs相对提升最高达10.4%）。  
- 在保持实时帧率（如RT-DETR R18仅从80FPS降至78FPS）的同时实现了精度增益。  
- 在分割任务上也验证了泛化能力，证明FINE并非仅为检测设计。

## 7. 优点

1. **轻量即插即用**：可无缝嵌入多种检测器，无需修改训练配置，附加开销极低。  
2. **有效的语义对齐**：通过AATS解决了跨层级ERF不对齐问题，实现了可靠的跨层注意力。  
3. **残差调制保留空间细节**：避免粗尺度信息直接替换细粒度特征，在小目标等定位敏感任务中表现好。  
4. **通用性强**：不仅在目标检测，在实例分割、语义分割、全景分割上也有提升。  
5. **分析深入**：通过热力图、ERF分析、错误分析等多种方式解释了提升来源。

## 8. 不足与局限

1. **大目标检测提升有限**：部分模型在大目标AP上略有波动（如YOLOv12-S），作者归因为统计噪声，但未在更广泛尺度上验证稳定性。  
2. **训练细节不完整**：未报告训练时长、学习率曲线等，复现可能需依赖官方代码。  
3. **应用场景限制**：当前FINE仍工作在骨干-颈部分离的范式下，未来可将跨层注意力直接整合到特征提取过程中。  
4. **未在超大规模模型或极端小目标数据集上充分测试**：仅在COCO和VisDrone上评估，缺乏对更复杂或低分辨率场景的覆盖。  
5. **对网络结构假设**：依赖特征通道统一或额外1×1卷积对齐通道，对某些YOLO架构需额外投影（附录C），可能轻微增加复杂度。

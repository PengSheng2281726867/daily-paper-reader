## 摘要
随着大语言模型数据集扩展到数万亿token，数据选择已成为过滤无信息噪声和构建自适应学习轨迹的关键前沿。除了静态启发式过滤外，用于LLM训练的先进数据选择方法主要遵循两种范式，每种都有根本性局限。基于影响的方法提供了原则性的双层目标，但需要难以处理的黑塞逆矩阵计算；而基于过量损失的方法计算效率高，但依赖于静态参考模型，在训练过程中与不断演化的代理模型不匹配。我们提出BLADE（双层自适应数据选择），一种无黑塞矩阵的数据选择框架。BLADE通过拉格朗日乘数将基于影响的方法背后的双层优化问题重新表述为带惩罚的单层目标，避免了黑塞逆矩阵计算，同时揭示了与基于过量损失的数据选择之间的原则性联系。所得目标恢复了过量损失形式，但用与训练保持同步的动态参考模型取代了静态参考模型。理论上，我们证明了这种带惩罚的表述保证了一阶收敛性。为了高效在线批次选择，我们将BLADE实例化为无记忆随机块坐标Frank-Wolfe算法。大量实验表明，BLADE持续优于最先进的数据选择基线，为LLM训练提供了一种实用方案。

## Abstract
As Large Language Model (LLM) datasets scale to trillions of tokens, data selection has emerged as a critical frontier to filter out uninformative noise and construct adaptive learning trajectories. Beyond static heuristic filtering, advanced data selection methods for LLM training largely follow two paradigms, each with fundamental limitations. Influence-based methods provide principled bi-level objectives but require intractable inverse-Hessian computations, while excess-loss methods are computationally efficient but rely on a static reference model that becomes misaligned with the evolving proxy model during training. We propose BLADE (Bi-Level Adaptive Data sElection), a Hessian-free framework for data selection. BLADE reformulates the bi-level optimization problem underlying influence-based methods as a penalized single-level objective via Lagrange multipliers, avoiding inverse-Hessian computation while revealing a principled connection to excess-loss based data selection. The resulting objective recovers an excess-loss form but replaces the static reference model with a dynamic one that stays synchronized with training. Theoretically, we prove that this penalized formulation guarantees first-order convergence. For efficient online batch selection, we instantiate BLADE as a memoryless randomized block-coordinate Frank-Wolfe algorithm. Extensive experiments show that BLADE consistently outperforms state-of-the-art data selection baselines, providing a practical recipe for LLM training.

# BLADE：可扩展的双层自适应数据选择方法——论文详细总结

## 1. 核心问题与整体含义（研究动机和背景）
大语言模型（LLM）的预训练数据集规模已达万亿token级别，直接使用所有数据训练会导致边际收益递减，因为低质量、噪声或高度冗余的token会降低模型推理能力。因此，**数据选择**——即挑选信息量高、信号强的训练样本以最优提升性能——成为关键问题。

现有模型感知的数据选择方法主要分为两个范式：
- **基于影响函数的方法**：通过估计每个训练样本对验证损失的影响（如 LESS、G-DIG、MATES），理论上严谨，但需要计算逆Hessian矩阵，计算代价极高，难以扩展到LLM规模。
- **基于超额损失的方法**：通过代理模型与参考模型之间的损失差来评分（如 RHO-1、可学习性优先训练），计算高效，但参考模型通常预先训练且固定不变，随着训练进行，参考模型与不断演化的代理模型之间出现失配，导致选择信号陈旧。

两种范式之间的关系尚不明确：影响函数方法有理论基础却难以扩展，超额损失方法高效却缺乏与双层数据选择目标的原理性联系。

**BLADE 的目标**：构建一个桥接上述两个范式的框架，既保留影响函数方法的原则性双层目标，又具备类似于超额损失方法的可扩展性和在线效率。

## 2. 方法：核心思想、关键技术细节与算法流程

### 2.1 双层问题形式化
- 给定训练数据集 $D_{train} = \{x_n\}_{n=1}^N$（以token为选择单元），定义选择向量 $\alpha \in [0,1]^N$ 且满足 $\sum \alpha_n = \gamma N$（$\gamma$ 为保留比例）。
- 内层问题：$w(\alpha) = \arg\min_w L_{train}(\alpha, w) = \frac{1}{\gamma N} \sum \alpha_n L_n^{train}(w)$。
- 外层问题：$\min_{\alpha \in A} L_{val}(w(\alpha))$。

### 2.2 BLADE：无Hessian的双层数据选择
**核心思想**：将内层最优性条件作为约束，通过拉格朗日乘子将其转化为单层惩罚目标：
$$ \min_{\alpha \in A, w} H_\lambda(\alpha, w) = L_{val}(w) + \lambda \left[ L_{train}(\alpha, w) - \min_u L_{train}(\alpha, u) \right]. $$
该公式完全避免了逆Hessian计算，并揭示了与超额损失选择的联系。

**算法流程（Algorithm 1）**：
1. **初始化**：代理模型 $u$ 和参考模型 $w$ 同步起点。
2. **更新代理模型 $u$**：每步对选定数据进行一次梯度下降：
   $$ u^{(t+1)} = u^{(t)} - \eta_u \nabla_u L_{train}(\alpha^{(t)}, u^{(t)}). $$
3. **周期性更新参考模型 $w$**：每 $\tau$ 步（一个 episode）从当前 $u$ 开始，运行 $K$ 步梯度下降，同时使用验证损失和训练损失：
   $$ w_{k+1}^{(e)} = w_k^{(e)} - \eta_w \left[ \nabla_w L_{val}(w_k^{(e)}) + \lambda \nabla_w L_{train}(\alpha^{(e\tau)}, w_k^{(e)}) \right]. $$
   更新后的 $\tilde{w}^{(e)}$ 用于后续 $\tau$ 步的选择。
4. **更新选择权重 $\alpha$**：通过Frank-Wolfe算法，最优方向为：
   $$ s = \arg\min_{\alpha \in A} \langle \Delta, \alpha \rangle, \quad \Delta_n = L_n^{train}(\tilde{w}) - L_n^{train}(u). $$
   取 $\eta_\alpha=1$ 时得到封闭的Top-K离散选择：$\alpha_n^{(t+1)} = \mathbb{I} \left[ -\Delta_n^{(t)} \in \text{Top-}\gamma N \right]$。
5. **在线批量实现**：采用无记忆随机块坐标Frank-Wolfe，每步均匀采样候选批次 $B_{cand}^{(t)}$，在该子集内进行Top-$\gamma |B_{cand}|$ 选择，实现高效的在线自适应课程学习。

**理论保证（Theorem 1）**：在标准正则性假设（PL条件、光滑性、有界Hessian）下，BLADE以 $O(T^{-1/4})$ 的速率收敛到原双层目标的一阶平稳点。

### 2.3 与现有方法的统一
- **统一影响函数方法**：BLADE的惩罚公式隐式等价于影响函数 $-\nabla_w L_n^{train}(w)^T H^{-1} \nabla_w L_{val}(w)$，但无需显式计算Hessian。
- **改进超额损失方法**：BLADE的评分准则 $L_n^{train}(\tilde{w}) - L_n^{train}(u)$ 恢复为超额损失形式，但参考模型 $\tilde{w}$ 是动态同步的，而非静态固定。

## 6. 主要结论与发现
- BLADE在**所有场景**下一致优于所有基线。例如在域转移（TinyLlama-1.1B）中平均准确率29.7%，比RHO-1高4.0%，比Base-CT高13.0%；在通用预训练中平均准确率51.73%，比RHO-1高1.15%。
- 动态参考模型是关键：相比RHO-1的静态参考导致损失分布坍缩，BLADE通过周期同步保持$\Delta>0$的右尾分布，持续选择具有正边际效用的token。
- 超参鲁棒：$\tau$ 在500～2000范围内性能稳定；保留比例 $\gamma=0.6$ 最优，过高（0.8）引入噪声导致性能下降。

- 跨模型迁移有效但非最优：将1.1B模型选择的token用于7B训练，仍优于随机和Base-CT，但低于在7B上直接运行BLADE，说明最优选择具有模型容量依赖性。

## 7. 优点
- **理论严谨**：基于拉格朗日乘子将双层优化转化为单层惩罚问题，消除逆Hessian；提供一阶收敛性保证（$O(T^{-1/4})$）。
- **桥接两种范式**：既能获得影响函数方法的原理支撑，又具备超额损失方法的计算高效性。
- **计算效率高**：复杂度与RHO-1相近（$O(\frac{2}{3}\gamma + \frac{\gamma K}{\tau} + \frac{2}{3}) C T$），远低于MATES；峰值显存未增加。
- **自适应动态选择**：参考模型与代理模型周期性同步，自动调整选择策略以适应模型能力变化。
- **实用性**：在线批量实现（随机块坐标Frank-Wolfe），可直接融入标准LLM训练流水线；超参鲁棒。

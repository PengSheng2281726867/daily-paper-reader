## 摘要
我们提出Casper3D，一种轻量级概率框架，用于将嘈杂的多视角2D基础模型嵌入转化为潜在3D语义表示。我们将视图级语义特征建模为底层3D语义状态的嘈杂观测，并使用基于集合的变分模型（在推理多视图时结合相对位姿）来推断该状态。Casper3D通过从新视角预测留存语义观测进行训练，同时与视觉和文本语义空间保持对齐，以实现开放词汇的3D理解。该框架与骨干网络无关，适用于语言对齐和自监督嵌入。实验表明，Casper3D产生的3D语义比简单的多视图池化更稳定，特别是在模糊和嘈杂的设置中。

## Abstract
We present Casper3D, a lightweight probabilistic framework for converting noisy multi-view 2D foundation-model embeddings into a latent 3D semantic representation. We model view-level semantic features as noisy observations of an underlying 3D semantic state and infer this state with a set-based variational model that incorporates relative pose during multi-view reasoning. Casper3D is trained by predicting held-out semantic observations from novel viewpoints, while remaining aligned with visual and text semantic spaces for open-vocabulary 3D understanding. The framework is backbone-agnostic and applies to both language-aligned and self-supervised embeddings. Experiments show that Casper3D produces more stable 3D semantics than simple multi-view pooling, especially in ambiguous and noisy settings.

## 论文中文总结

### 1. 核心问题与整体含义（研究动机和背景）

- **问题**：从多视图 2D 基础模型（如 CLIP、DINO）中提取的 3D 语义特征通常不稳定。现有方法大多采用“提升-聚合”（lift-and-aggregate）策略，如平均池化或加权投票，这些方法将 2D 观测特征直接混合为 3D 特征，隐式假设 3D 特征可由 2D 观测的加权均值近似。然而，2D 特征受视角、遮挡、光照、图像上下文等因素影响，是 **有噪的、视角依赖的观测**，而非 3D 语义的完美表征。
- **动机**：作者将 3D 特征提升视为一个 **贝叶斯逆问题**：每个 3D 点拥有一个潜在的、视角不变的语义状态 \(z_i\)，而不同视角下的 2D 特征 \(x_{ij}\) 是该状态的有噪观测。目标是从多视角观测 \(C_i\) 中推断出后验分布 \(p(z_i \mid C_i)\)，而不是简单地合并观测值。
- **背景**：该工作属于开放词汇 3D 场景理解领域，旨在利用 2D 视觉-语言模型（VLM）为 3D 点赋予语义，无需 3D 标注。

### 2. 方法论：核心思想、关键技术细节、公式

- **核心思想**：
  - 每个 3D 点 \(p_i\) 对应一个未观测的潜在变量 \(z_i \in \mathbb{R}^d\)（代表其视角不变语义状态）。
  - 视角 \(T_j\) 下的 2D 观测 \(x_{ij}\) 由 \(p_\theta(x \mid z_i, T_j)\) 产生（噪声观测模型）。
  - 目标：从观测集 \(C_i = \{(x_{ij}, T_j)\}\) 推断后验 \(p(z_i \mid C_i)\)。由于精确后验难解，使用变分推断近似 \(q_\phi(z_i \mid C_i) = \mathcal{N}(\mu_i, \text{diag}(\sigma_i^2))\)。
- **关键技术细节**：
  - **编码器（Encoder）**：基于集合的变分模型，输入无序的观测集合 \(C_i\)，输出后验参数 \((\mu_i, \sigma_i)\)。采用 Pose-aware Attention Block (PAB)，引入 **相对位姿** \(T_{l \to k}\) 作为注意力偏置，实现置换等变性。
  - **解码器（Decoder）**：从后验采样 \(z_i\)，结合目标视角 \(T_k\)，预测该视角下的观测 \(\hat{x}_{ik} = \text{Decoder}_\theta(z_i, T_k)\)。
  - **训练目标**：通过 **留出视图预测**（held-out view prediction）进行自监督训练。对每个点，将观测分为上下文集 \(C_i\) 和留出集 \(H_i\)。编码器从 \(C_i\) 推断 \(z_i\)，解码器预测 \(H_i\) 中的观测，优化负 ELBO：
    \[
    \mathcal{L}_{\text{ELBO}}(i) = \underbrace{-\mathbb{E}_{z_i \sim q_\phi(\cdot | C_i)} \left[ \sum_{k=1}^{N_i} \log p_\theta(x_{ik} | z_i, T_k) \right]}_{\text{留出预测损失}} + \beta \, \text{KL}[q_\phi(z_i | C_i) \parallel p(z_i)]
    \]
    其中留出预测损失使用余弦损失实现。
  - **额外损失**：
    - **文本对齐损失** \(\mathcal{L}_{\text{text}}\)：使推断出的 3D 特征与文本原型兼容（用于开放词汇任务）。
    - **弱锚定损失** \(\mathcal{L}_{\text{avg}}\)：将最终 3D 特征锚定到平均池化特征，防止漂移。
  - **网络架构**：PAB 块包含相对位姿编码，与 Set Transformer 类似但加入了位姿信息。编码器由多个 PAB 堆叠后接 PMA（Pooling by Multihead Attention）实现置换不变性。
- **算法流程**（文字说明）：
  1. 输入：点云中每个点的多视图观测集（特征+位姿）。
  2. 编码器处理上下文观测，输出后验分布参数。
  3. 从后验采样潜在变量 \(z_i\)。
  4. 解码器以 \(z_i\) 和留出位姿为输入，预测留出视图特征。
  5. 通过对比预测特征与真实特征计算损失，优化模型。
  6. 推理时，所有观测作为上下文，使用后验均值 \(\mu_i\) 经投影得到最终 3D 特征 \(e_i\)。

### 6. 主要结论与发现

- **Casper3D 显著优于所有零样本 2D 融合方法**，在 ScanNet 上 mIoU 64.6% vs OpenScene-2D 50.0%，Matterport3D 上 50.4% vs 32.3%。
- **无需 3D 训练即接近甚至超越部分需 3D 训练的方法**，如 ScanNet 上 Casper3D 64.6% vs OpenScene+3D 52.9%、OV3D 57.3%。
- **结合 3D 蒸馏后进一步增益**（Ours+3D Distillation 在 ScanNet 上 65.8%），缩小与全监督（72.0%）的差距。
- **对稀疏视图更鲁棒**：在仅使用 4 个视图时，Casper3D 仍保持 56.0% mIoU，而平均池化仅 20.1%。
- **轻量高效**：单次推理仅需 14.4 ms，参数量 35.6M，显存 150 MB。

### 7. 优点

- **概率建模**：将问题置于贝叶斯框架下，自然处理观测噪声和不确定性，输出后验分布（均值和方差），提供信心估计。
- **视角不变性**：通过留出视图预测迫使潜在变量学习视角不变语义，避免编码视角伪影。
- **骨干无关**：适用于各种 2D 基础模型（CLIP, DINO, OpenSeg 等），可随骨干升级自动提升。
- **自监督训练**：无需任何 3D 标注，仅需多视图 RGB-D 数据和预训练 2D 特征。
- **集成相对位姿**：PAB 模块利用相对位姿增强注意力，提升多视图推理能力。
- **轻量与高效**：相比 3D 蒸馏方法，Casper3D 本身计算开销极低，可作为独立模块或前置处理。

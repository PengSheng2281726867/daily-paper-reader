## 摘要
Transformer的计算复杂度随令牌数量呈二次增长，这严重限制了视觉模型的效率，尤其是近期基于ViT的基础模型在密集预测任务中的表现。实例分割作为遥感领域典型的密集视觉预测任务，面临类似挑战。本文受大型语言模型中知识蒸馏最新进展的启发，提出RS4D——一种具有线性计算复杂度的新型遥感实例分割方法，通过蒸馏状态空间建模解决了长序列建模效率低下的问题。我们提出了一种自适应噪声与掩码知识蒸馏训练方法，用于预训练轻量级SSM骨干网络，有效地将庞大的自注意力空间中的知识压缩到紧凑的密集线性状态空间中。我们还设计了一种基于该轻量级视觉编码器的遥感图像实例分割架构，探索了三种不同骨干网络和两种分割头的变体。在包括SSDD、WHU和NWPU在内的多个基准数据集上进行了大量实验。与基于ViT的方法相比，我们提出的SSM骨干网络在参数数量上减少了8倍，FLOPs减少了9倍，同时保持与基于ViT和CNN的实例分割方法相当或更高的精度。实现代码已公开于https://github.com/QinzheYang/RS4D。

## Abstract
The computational complexity of Transformers scales quadratically with the number of tokens, which significantly constrains the efficiency of vision models, particularly recent ViT-based foundation models in dense prediction tasks. Instance segmentation, a typical dense visual prediction task in the remote sensing field, faces similar challenges. In this paper, inspired by the recent advances of knowledge distillation in large language models, we introduce RS4D - a new remote sensing instance segmentation method with linear computational complexity, which addresses the inefficiency of long sequence modeling through distilled state space modeling (SSM). We propose an adaptive noise and masking knowledge distillation training method for pre-training lightweight SSM backbones, which effectively compresses knowledge from the vast self-attention space into a compact, dense linear state space. We also design a remote sensing image instance segmentation architecture based on this lightweight visual encoder, where we explore variants of three different backbones and two segmentation heads. Extensive experiments are conducted on multiple benchmark datasets, including SSDD, WHU, and NWPU. Compared to ViT-based approaches, our proposed SSM backbone achieves an 8x reduction in parameters and a 9x reduction in FLOPs while maintaining comparable or superior accuracy to both ViT- and CNN-based instance segmentation methods. The implementation codes have been publicly available at https://github.com/QinzheYang/RS4D.

#### 1. 核心问题与整体含义（研究动机和背景）

-   **核心问题**：遥感图像的实例分割本质上是密集预测任务，需要处理高分辨率的长序列数据。基于Transformer的视觉基础模型（如SAM）在处理这类任务时，其自注意力机制的计算复杂度与序列长度成**二次方（O(n²)）** 关系，导致计算量庞大、显存消耗高，严重限制了在处理高分辨率遥感图像时的实用性和效率，尤其是在资源受限的实时应用和边缘设备（如星上处理）上。这也使得这类模型的参数数量通常较大。


-   **整体含义**：为了解决Transformer的效率瓶颈，本文提出将**状态空间模型**和**知识蒸馏**两种技术相结合，旨在构建一个既具有线性计算复杂度、又能保留基础模型强大分割先验知识的轻量级遥感实例分割框架。这在追求模型性能与效率的平衡方面具有重要的现实意义。

#### 2. 核心方法论

-   **整体思想**：提出RS4D框架，通过**知识蒸馏**，将复杂的基于ViT的SAM教师模型的分割知识，转移到一个轻量级的、基于状态空间模型的SSM学生骨干网络中。学生模型在推理时具有**线性计算复杂度**，从而实现高效分割。

-   **关键技术细节**：
    1.  **SSM 骨干网络设计**：提出了三种轻量级SSM学生骨干架构：
        -   **VanillaMamba**: 直接堆叠12个Vision Mamba (ViM) 块，结构最简单。
        -   **TransMamba**: 混合Mamba-Transformer架构，用SSM块替换Transformer中的注意力层，但保留了MLP和残差连接（将ReLU替换为GELU以提高鲁棒性），旨在利用SSM的线性复杂度同时保留Transformer的深层特征融合能力。
        -   **ScanningMamba**: 默认为最佳方案。采用**双向扫描**机制（前后向），分别处理原始顺序和反向顺序的序列，然后平均结果，以获得更完整的二维空间上下文特征，并加深至24层以承载更多的先验知识。
    2.  **自适应噪声与掩码知识蒸馏**：
        -   利用无标签的SA-1B数据集进行预训练。
        -   **对输入进行干扰**：训练时，对输入学生模型的像素随机添加**掩码**和**噪声**。噪声的幅度（`α e`）会根据学生模型前一epoch的输出特征图的**最大绝对值**进行自适应调整，以保证训练稳定性。公式化表示为：`X' = mask(X_input) + α e * ϵ`，其中`α e = min(max(|Y e-1|), λ const)`。
        -   **损失函数**：使用教师模型（SAM编码器）和学生模型输出之间特征的 **L1距离**作为蒸馏损失，使得教师模型的特征空间知识（Shape & Structure）被压缩到学生模型中。
    3.  **高效分割架构**：
        -   **Neck（颈部）**：设计了一个轻量级的**多级特征融合颈**，由下采样层、级联残差模块和简易FPN组成。它从SSM骨干的特定中间层选择性提取并融合特征，以增强对遥感图像中小目标的特征提取能力，同时保持计算量极小。
        -   **Head（头部）**：提出了两种分割头：
            -   **Box-based Head**：遵循经典的Mask R-CNN范式，使用RPN和RoI头，最后通过FCN进行掩码预测。损失函数为 `L = L_rpn-cls + L_rpn-bbox + L_roi-cls + L_roi-bbox + L_fcn-mask`。
            -   **Prompt-based Head**：**创新结构**。将RoI特征映射为**隐式的提示令牌**，输入到SAM风格（‘prompt’）的提示编码器和掩码解码器中生成最终掩码，从而替换掉原本的FCN。损失函数为 `L = L_rpn-cls + L_rpn-bbox + L_roi-cls + L_roi-bbox + L_decoder-mask`。

## 摘要
近期，视觉Transformer（ViT）在人脸识别（FR）中的进展已超越了标准的CLS令牌范式。在该范式中，一个特殊的分类令牌（CLS）被附加到图像块嵌入之前，并作为下游任务的输入表示使用。另一种方法，即拼接图像块嵌入（CPE），则利用所有图像块令牌，将它们拼接成一个单一向量，然后投影为紧凑的人脸表示。与基于CLS的方法相比，CPE已被证明能提高识别性能，但我们对注意力图进行定性分析时发现，存在一些限制其可解释性的伪影。为了解决这个问题，我们引入了注册令牌，这些可学习令牌被拼接到初始图像块嵌入中，并通过ViT编码器块联合处理。与基线ViT相比，这种机制已被证明能产生更结构化、更可解释的注意力图。我们通过实证表明，这些伪影在各种ViT骨干网络（包括小型和大型模型）中一致出现，而引入注册令牌能有效减轻它们。添加四个或八个注册令牌显著增强了可解释性，其中八个注册令牌提供了最高的验证准确率和最平滑的注意力结构。我们最终得到的模型ViT-8R，对应一个基于CPE的ViT-B架构，并增加了八个注册令牌，在大型IJB-B和IJB-C基准测试中，在所有基于ViT的人脸识别模型中达到了最先进的性能。此外，与基线模型相比，ViT-8R产生了明显更清晰的注意力图，为模型的注意力行为提供了更深入的洞察（https://github.com/TaharChettaoui/ViT-FR-Registers）

## Abstract
Recent advances in Vision Transformers (ViTs) for face recognition (FR) have moved beyond the standard CLS-token paradigm. In this paradigm, a special classification token (CLS) is prepended to the patch embeddings and used as a representation of the input for downstream tasks. An alternative approach, Concatenated Patch Embeddings (CPE), instead leverages all patch tokens by concatenating them into a single vector, which is then projected into a compact face representation. CPE has been shown to improve recognition performance in comparison to CLS-based ones, but our qualitative analysis of attention maps showed the presence of artifacts that limit their interpretability. To address this issue, we incorporate register tokens, learnable tokens concatenated to the initial patch embeddings, and processed jointly through the ViT encoder blocks. This mechanism has been shown to produce more structured and interpretable attention maps compared to baseline ViT. We empirically demonstrate that these artifacts consistently appear across various ViT backbones, including small and large models, and that introducing register tokens effectively mitigates them. Adding four or eight registers significantly enhances interpretability, with eight registers providing the highest verification accuracies and smoothest attention structures. Our resulting model, ViT-8R, corresponds to a CPE-based ViT-B architecture augmented with eight register tokens achieves state-of-the-art performance among ViT-based FR models on large-scale IJB-B and IJB-C benchmarks. Also, ViT-8R produces substantially clearer attention maps compared with the baseline model, which offer deeper insight into the model's attention behavior (https://github.com/TaharChettaoui/ViT-FR-Registers)

### 1. 核心问题与整体含义（研究动机和背景）

- **背景**：视觉Transformer（ViT）在人脸识别（FR）中逐渐取代传统CNN和标准CLS-token范式。近期性能更优的方法是**拼接图像块嵌入（CPE）**，它丢弃CLS令牌，将所有图像块令牌拼接后投影为紧凑人脸表示。
- **问题**：尽管CPE方法识别性能更好，但其注意力图存在明显的**伪影**（artifacts），尤其是在低信息背景区域，严重限制了模型的可解释性，且此前的研究未在FR场景中专门解决这一问题。
- **目标**：借鉴Darcet等人[12]在通用图像分类中提出的**注册令牌（register tokens）**，将其引入CPE-based ViT用于人脸识别，旨在消除注意力伪影，同时提升识别性能。最终期望获得更清晰、可解释的注意力图，并验证其在FR任务上的有效性。

### 2. 方法论：核心思想、关键技术细节与流程

- **核心思想**：在ViT的初始图像块嵌入序列后添加一组**可学习的注册令牌**（register tokens），它们与图像块令牌一起经过所有Transformer编码器层，但**不参与最终的人脸特征表示**（在输出时丢弃）。注册令牌作为“内部存储”，吸收模型在低信息区域产生的冗余计算，从而避免伪影的产生。
- **关键技术细节**：
  - **架构**：采用CPE-based ViT（无CLS令牌），输入为图像块嵌入序列 `[z1, z2, ..., zN]`。注册令牌 `[r1, r2, ..., rR]` 被拼接到该序列之后，构成输入 `Z_reg = [z1, ..., zN, r1, ..., rR]`。
  - **处理**：该序列通过标准Transformer编码器（多头自注意力MSA + 多层感知机MLP），注册令牌参与所有计算，但不添加位置编码（因为注册令牌本身不携带位置信息）。
  - **特征提取**：编码器输出后，仅保留所有图像块令牌（patch tokens），将它们拼接并通过一个MLP投影到最终的人脸表示维度，注册令牌直接丢弃。
  - **损失函数**：采用margin-penalty softmax损失（如CosFace、ArcFace、AdaFace），进行多分类训练，提升特征判别力。
- **公式流程**（文字说明）：
  1. 图像分割为N个9×9的patch，线性投影到D维嵌入。
  2. 生成R个可学习注册令牌（维度D），与N个patch令牌拼接。
  3. 加入位置编码（仅针对patch令牌），送入L层Transformer编码器。
  4. 在每一层，所有令牌（包括注册令牌）参加多头自注意力。
  5. 最后一层输出后，提取所有patch令牌，拼接为N*D向量，通过MLP降维到D作为最终人脸特征。
  6. 注册令牌不用于最终表示，但在编码过程中起了缓冲作用。

### 6. 主要结论与发现

1. **CPE-based ViT在人脸识别中普遍存在注意力伪影**，且在所有骨干（包括Tiny和Small）中都出现，这与原始注册令牌论文[12]中的观察不同（原论文中小模型伪影较少）。
2. **注册令牌能有效消除伪影**：当注册令牌数量达到4或8时，注意力图变得结构化且集中在面部关键区域（眼、鼻、嘴）；8个注册令牌提供最佳性能。
3. **ViT-8R（ViT-B + 8 registers）在IJB-B和IJB-C上达到ViT-based FR的SOTA**，在MS1MV2、MS1MV3、WebFace4M训练下均取得领先；在小型基准上取得非常有竞争力的结果。
4. 注册令牌带来的计算开销很小（ViT-B增加1.33 GFLOPs），可忽略不计。
5. 注册令牌不依赖特定损失函数（CosFace、ArcFace、AdaFace均有效），且可迁移至其他ViT变体（如KP-RPE）。
6. 注意力定量指标MAWS随注册令牌增加而提高，但16个注册令牌时略有下降，说明存在最优数量。

### 7. 优点

- **创新性**：首次系统性地将注册令牌引入人脸识别ViT，解决了CPE方法可解释性差的核心问题。

- **实用性**：注册令牌实现简单、计算开销小，可直接应用于现有CPE-based ViT流水线，无需重大架构修改。

- **SOTA性能**：在大规模基准（IJB-B/C）上超过此前所有ViT-based FR方法，同时保持小型基准的竞争力。

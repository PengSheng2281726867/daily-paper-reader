## 摘要
视觉Transformer（ViTs）已成为视觉表示学习的主导架构，提供了极其强大且广泛可重用的骨干特征。然而，由于全局自注意力的二次成本，ViT通常操作在相对较小的块标记网格上，这给语义分割和深度估计等密集预测任务带来了持续瓶颈。这推动了任务无关的特征上采样器的发展。虽然最近的最先进方法能产生视觉上清晰的密集表示，但它们依赖浅层图像编码器进行引导上采样，可能导致特征泄漏、碎片化和模糊。我们引入了ViT-Up，一种隐式特征上采样框架，用从中间ViT隐藏状态逐层构建查询的方式替代外部图像引导。这能够在任意连续图像坐标上进行特征预测，同时保持与骨干特征空间的对齐。实验表明，ViT-Up在密集预测和语义对应任务上始终优于最先进的图像引导上采样器。在DINOv3-S+上，ViT-Up在Cityscapes上相比先前方法提升了+2.07 mIoU，在SPair-71k上提升了+4.17 PCK@0.10。使用更大的DINOv3-B骨干时，这些增益分别增加到+3.36 mIoU和+8.09 PCK@0.10，表明ViT-Up随着骨干容量的增大而扩展性良好。

## Abstract
Vision Transformers (ViTs) have become a dominant architecture for visual representation learning, providing exceptionally strong and broadly reusable backbone features. However, ViTs are commonly operated on relatively small patch-token grids due to the quadratic cost of global self-attention, which creates a persistent bottleneck for dense prediction tasks such as semantic segmentation and depth estimation. This has motivated the development of task-agnostic feature upsamplers. While recent state-of-the-art methods produce visually sharp dense representations, their reliance on shallow image encoders for guided upsampling can introduce feature leakage, fragmentation, and blur. We introduce ViT-Up, an implicit feature upsampling framework that replaces external image guidance with layer-wise query construction from intermediate ViT hidden states. This enables feature prediction at arbitrary continuous image coordinates while preserving alignment with the backbone feature space. Experiments demonstrate that ViT-Up consistently outperforms state-of-the-art image-guided upsamplers across dense prediction and semantic correspondence. On DINOv3-S+, ViT-Up improves over prior methods by up to +2.07 mIoU on Cityscapes and +4.17 PCK@0.10 on SPair-71k. With the larger DINOv3-B backbone, these gains increase to +3.36 mIoU and +8.09 PCK@0.10, demonstrating that ViT-Up scales favorably with backbone capacity.

# 论文总结：ViT-Up: Faithful Feature Upsampling for Vision Transformers

## 1. 核心问题与整体含义（研究动机与背景）

- **问题**：Vision Transformer（ViT）因全局自注意力的二次复杂度，通常只在较小的 patch 网格（如 14×14 或 28×28）上提取特征。这导致 dense prediction 任务（语义分割、深度估计）面临分辨率瓶颈。
- **现有方法局限**：主流任务无关上采样方法（如 JAFAR、AnyUp、NAF、UPLiFT）依赖外部浅层图像编码器提供高分辨率引导，虽然视觉上清晰，但存在**特征泄漏、碎片化和模糊**问题。因为浅层编码器缺乏语义理解能力，容易将视觉相似但语义不同的区域混在一起。
- **核心思路**：利用 ViT 自身的层次结构——浅层保留空间细节，深层凝聚抽象语义——直接从中间隐藏状态构建高分辨率特征，避免外部图像引导带来的语义错位，实现**与骨干特征空间对齐的忠实上采样**。

## 2. 方法论

### 2.1 整体框架
- 将特征上采样视为**坐标条件特征预测**：给定任意连续图像坐标 \( x_q \)，ViT-Up 从低分辨率 ViT 中间隐藏状态 \( H_{l[t]} \) 预测对应的高分辨率特征。
- 输出高分辨率特征图可通过在密集坐标网格上独立查询获得，支持任意输出分辨率。

### 2.2 关键组件

| 组件 | 功能 | 技术实现 |
|------|------|----------|
| **查询嵌入（Query Embedding）** | 初始化查询表示 | 重用 ViT patch embedding（卷积层），对输入图像放大后获得 224×224 的 patch 网格，在查询坐标处双线性插值得到 \( q_0 \in \mathbb{R}^C \) |
| **ViT-Up 块（T=6）** | 逐步细化查询特征 | 每个块依次经过：过渡 MLP（残差）→ 交叉窗口多头注意力（连续 2D RoPE 位置编码，使用 NATTEN 加速）→ FeatX → 融合 → 残差融合 MLP |
| **FeatX** | 恢复 sub-token 细节 | 提取最近邻 patch token 的特征，编码相对偏移 \( \Delta x \)，通过 FiLM 调制和 MLP 得到 sub-token 特征 |
| **骨干适应** | 提供额外容量而不全微调 | 对 patch embedding 和注意力投影施加 LoRA（rank=16，scale=32，dropout=0.05） |
| **输出解码器** | 映射回 ViT 特征空间 | 单层 MLP（LayerNorm + 线性投影） |

## 7. 优点

- **方法创新性**：首次系统利用 ViT 层次结构进行隐式上采样，避免外部图像引导带来的语义错位。
- **特征保真度高**：通过多尺度蒸馏和逐层细化，保持与骨干特征空间对齐，在对应任务上效果尤为突出。
- **实用灵活**：查询可独立分批处理，支持在内存受限场景下以小幅时间开销换取低显存；支持任意输出分辨率。
- **实验全面**：涵盖密集预测、语义/几何对应、特征保持、骨干缩放，消融逻辑清晰，对比基线均为官方或最优权重。

## 8. 不足与局限

- **信息瓶颈**：最终上采样质量仍受限于隐藏状态编码的信息。虽然利用中间层缓解，但超出 token 范围的细节无法恢复。未来可通过输入更高分辨率隐藏状态（如 56×56）改善。
- **骨干耦合**：ViT-Up 需为每个骨干单独训练（因过渡 MLP、FeatX 依赖特定隐藏状态组织），不具有 zero-shot 跨骨干能力。不过一旦训练好，可复用全部下游任务。
- **训练计算开销**：参数较多（24.9M），训练需多尺度教师前向（3 次 ViT 推理 + 1 次学生推理），但训练仅 1 epoch，尚可接受。
- **附录 D 的局限**：在 DINOv2 上 ViT-Up 不如图像引导方法（因为 DINOv2 有位置伪影，ViT-Up 忠实重建但也保留了伪影），说明方法更适合特征质量好的现代骨干。
- **深度估计实验**：冻头探测中 ViT-Up 不占优（因特征统计偏移），但微调后胜出。论文已承认此局限并给出合理解释。

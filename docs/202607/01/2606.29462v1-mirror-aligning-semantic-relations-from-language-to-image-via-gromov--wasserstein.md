## 摘要
多模态大语言模型从其语言骨干中继承了丰富的关联先验，但在被要求将这些关系应用于视觉场景时往往失败。我们将这种失败追溯到一个结构盲点：基于投影的对齐训练每个视觉token携带正确的语义，但从未询问概念间的关系是否经历了从语言到视觉的跨越。为了解决这个问题，我们提出了MIRROR（通过基于最优传输的正则化从语言到视觉表示映射概念间关系），这是一种几何正则化框架，通过利用语言表示中编码的丰富关系结构，将关系先验从语言转移到视觉。具体来说，我们从提出的半逆Gromov-Wasserstein（SI-GW）问题中推导出一个替代损失函数，这是一个逆几何问题，旨在将视觉表示与语言导出的关系先验对齐。我们证明该公式具有唯一的闭式解，该解规定了由语言几何和跨模态耦合所隐含的理想视觉关系结构。该公式的结构还支持高效计算，使其适用于长token序列。在仅解码器Transformer中应用SI-GW需要仔细设计。我们引入了层、头部和token级别的针对性策略，以确保稳定提取，而无需额外参数或推理成本。MIRROR在保持通用视觉-语言任务性能的同时，改善了关系一致性。

## Abstract
Multimodal Large Language Models (MLLMs) inherit rich relational priors from their language backbones, yet often fail when asked to apply these relationships in visual contexts. We trace this failure to a structural blind spot: projection-based alignment trains each visual token to carry the right semantics, but never asks whether the relationships between concepts survive the crossing from language to vision. To address this, we propose MIRROR (Mapping Inter-concept Relations from language to visual Representation via Optimal-transport-based Regularization), a geometric regularization framework that transfers relational priors from language to vision by exploiting the rich relational structure encoded in language representations. Specifically, we derive a surrogate loss from the proposed Semi-Inverse Gromov-Wasserstein (SI-GW) problem, an inverse geometric problem that aligns visual representations with language-derived relational priors. We show that this formulation admits a unique closed-form solution that prescribes the ideal visual relational structure implied by language geometry and cross-modal coupling. The structure of the formulation also enables efficient computation, making it applicable to long token sequences. Applying SI-GW inside decoder-only Transformers requires careful design. We introduce targeted strategies at the layer, head, and token levels to ensure stable extraction without additional parameters or inference cost. MIRROR improves relational consistency while preserving performance on general vision-language tasks.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：多模态大语言模型（MLLM）从语言骨干网络继承了丰富的语义关系先验（如空间配置、逻辑依赖），但在视觉上下文中应用这些关系时频繁失败。例如，模型在仅有文本时能正确推理“叶子茂盛”，但结合图像后反而被视觉表象误导得出错误答案。
- **根源**：现有投影式对齐（projection-based alignment）仅训练每个视觉 token 携带正确的语义，却从未显式要求**概念间的关系结构**在跨模态迁移中得到保持。这是一种“结构盲点”（structural blind spot）：token 级对齐保证了身份（identity）正确，但丢失了关系几何。
- **理论支撑**：表征趋同假说（Platonic Representation Hypothesis）和 LLM 视觉先验研究表明，文本因其符号性和组合性，编码了更结构化的语义几何。因此，语言可以作为“教师”来改善视觉的关系推理。

## 2. 方法论：核心思想、关键技术细节与算法流程

### 核心思想
- 将关系迁移形式化为一个**逆几何问题**：给定语言几何（文本 token 间距离矩阵 \(D_t\)）和跨模态耦合 \(C\)（由交叉注意力提供），求解理想的视觉几何 \(\hat{D}_v\)，使得在 \(C\) 下两个空间的成对距离结构保持一致。
- 提出**Semi-Inverse Gromov–Wasserstein (SI-GW)** 问题，并证明其有唯一闭式解：\(\hat{D}_v = (C^\top D_t C) \oslash (b b^\top)\)，其中 \(b = C^\top \mathbf{1}\)。该解可解释为条件期望：每个视觉 token 对的距离等于对应文本 token 对距离在耦合权重下的加权平均。
- 将 SI-GW 的闭式解作为目标，定义正则化损失 \(L_{\text{SI-GW}} = \| D_v - \hat{D}_v \|_F^2\)，叠加在标准语言建模损失上。

### 关键技术细节
- **距离度量构造**：从自注意力矩阵提取，对称化后取负对数变换 \(D = -\log(A + \varepsilon)\)，使高注意力映射为小距离。
- **稳定提取策略**（针对 decoder-only Transformer）：
  - **层解耦**：文本自注意 \(A_t\) 取自早期 LLM 层（关系纯净），视觉自注意 \(A_v\) 取自 ViT 最后一层，耦合 \(C\) 取自中间 LLM 层（平衡信息丰富度与集中度）。
  - **选择性头聚合**：计算每个交叉注意头的熵，保留 top‑k 低熵的头（注意力更集中），平均得到耦合 \(C\)。
  - **非语义 token 过滤**：根据位置和耦合质量对文本 token 施加软抑制，降低系统提示、占位符等噪声。
- **计算效率**：直接计算四重和复杂度 \(O(n_t^2 n_v^2)\)，利用闭式解降为 \(O(n_t^2 n_v + n_v^2 n_t)\)，可处理长序列。

### 算法流程
1. 前向传播获取 ViT 和 LLM 的所有注意力矩阵。
2. 解耦提取 \(A_v\)、\(A_t\)、各头交叉注意力 \(\{A_h\}\)。
3. 熵选择 top‑k 头，平均得到 \(C\)；对 \(A_t\) 进行非语义 token 过滤得到 \(\tilde{A}_t\)。
4. 计算 \(D_t = -\log(\tilde{A}_t + \varepsilon)\)，\(D_v = -\log(A_v + \varepsilon)\)。
5. 计算 \(\hat{D}_v = (C^\top D_t C) \oslash (b b^\top)\)，\(b = C^\top \mathbf{1}\)。
6. 计算 \(L_{\text{SI-GW}} = \| D_v - \hat{D}_v \|_F^2\)。
7. 总损失 \(L_{\text{total}} = L_{\text{LM}} + \lambda L_{\text{SI-GW}}\) 反向传播更新参数。

## 7. 优点
- **理论创新**：将关系跨模态对齐形式化为一个有闭式解的逆几何问题，方案优雅且可解释（条件期望意义）。
- **高效实用**：计算复杂度从 \(O(n_t^2 n_v^2)\) 降至 \(O(n_t^2 n_v + n_v^2 n_t)\)，无额外参数，推理零开销。
- **设计周密**：针对 decoder-only Transformer 的特性设计了层解耦、熵选择头、非语义过滤三套策略，确保梯度稳定。
- **实验扎实**：多模型、多任务、多难度层级评估，消融完整，结论一致性强。
